This is an example shopping page created following the Odin Project's shopping project. The details can be found here:
https://www.theodinproject.com/lessons/node-path-react-new-shopping-cart.

It is a simple page that allows users to browse a shop page and add items to their cart. It utilizes React Router for its client side routing. Tests are written with React Testing. All shopping data comes from the fake store api: https://fakestoreapi.com/.

A Cloudflare deployment of this project can be found here:
https://odin-shopping-cart-14t.pages.dev
